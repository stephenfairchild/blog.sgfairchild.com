# blog.sgfairchild.com

A place where I can rant and write case studies on software. This site is currently wrote with Remix.run and deployed on the Cloudflare edge network via Cloudflare Pages.

## Development

```sh
$ npm run dev
```

Open up [http://127.0.0.1:8788](http://127.0.0.1:8788) !

## Deployment

Push to `main` and the site will automatically deploy.
